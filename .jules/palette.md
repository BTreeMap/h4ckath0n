## 2024-05-20 - Keyboard shortcuts for textareas
**Learning:** Adding explicit keyboard shortcuts (Enter to send, Shift+Enter for newline) to AI chat textareas with visual `<kbd>` hints significantly improves user efficiency. Disabling the textarea during async operations prevents prompt mangling.
**Action:** Always implement Enter-to-submit on AI chat prompts, visually indicate the shortcut, and disable the input during streaming.
