## 2024-06-20 - Adding `aria-busy` to `Button` via Playwright
 **Learning:** When attempting to intercept and block network requests indefinitely (e.g. `page.route("...", lambda route: time.sleep(10))`) using the `sync_api` to capture visual loading states, the underlying asyncio loop crashes, preventing `page.screenshot` from executing. It is more robust to mock or manually inject the state change (e.g., `btn.evaluate("el => el.setAttribute('aria-busy', 'true')")`) for screenshot capture when testing micro-interactions than dealing with complex network timing constraints.
 **Action:** To verify transient DOM attributes without crashing Playwright, manually inject the state directly into the DOM using `page.evaluate` rather than simulating network latency via `time.sleep()`.
