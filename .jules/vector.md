## $(date +%Y-%m-%d) - Centralize scope parsing logic
**Learning:** The application extensively parsed, normalized, and formatted user authorization scopes via fragmented inline list comprehensions and standard string manipulations across the router, dependencies, and CLI files. In one place (`cli.py`), mutation to a `set` accidentally scrambled the deterministic scope ordering.
**Action:** Extract these into a set of pure utility functions (`parse_scopes`, `format_scopes`, `normalize_scope_list`) within a dedicated `scopes.py` helper to guarantee consistency, preserve order, and dramatically simplify test cases and inline validation. Ensure any test files interacting with original private helpers (like `_normalize_scopes` in `test_cli.py`) are refactored to test the new utilities correctly.
