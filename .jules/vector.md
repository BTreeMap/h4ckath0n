## 2024-05-15 - Centralized Authentication Scopes Normalization
**Learning:** The Python codebase previously duplicated logic for splitting, filtering, and joining comma-separated strings (specifically user scopes) across multiple files (cli.py, session_router.py, dependencies.py). This duplication also inconsistently handled edge cases like leading/trailing spaces and deduplication.
**Action:** Always prefer extracting text parsing, formatting, and normalization into pure helper functions (e.g., `parse_scopes`, `format_scopes`) placed in a centralized module (like `auth/scopes.py`). Use these helpers uniformly to enforce correctness and edge-case consistency (e.g., stripping whitespace, ignoring empty parts, deduplicating) across CLI, request dependencies, and API responses.
